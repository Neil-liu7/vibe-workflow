Workflow MCP 代码生成助手

基于 MCP 协议的智能工作流引擎，集成强大的 Prompt 管理和代码生成功能

TypeScript MCP Protocol AI Assistant GPL-3.0

核心功能特性

为开发者提供全方位的 AI 辅助开发体验

🔧
强大的工作流引擎

支持复杂的多步骤工作流定义，包括 Prompt 步骤、MCP 步骤和子工作流步骤

📝
智能Prompt管理

支持 YAML/JSON 格式的 Prompt 文件，动态参数替换，即时执行

🚀
代码生成和审查

内置代码生成、代码审查等专业 Prompt，支持多种编程语言

📚
自动文档生成

智能分析代码结构，自动生成高质量的技术文档和API文档

Prompt 管理功能

  • 文件管理: 支持 YAML 和 JSON 格式的 prompt 文件
  • 动态加载: 自动扫描和加载 prompts 目录中的所有文件
  • 参数替换: 支持 {{参数名}} 格式的动态参数替换
  • 即时执行: 直接执行 prompt 并获取结果

📖 doc_generator

工作流引擎架构

支持复杂的多步骤工作流定义和执行

1
Prompt Step

执行预定义的 Prompt 模板，支持参数替换

2
MCP Step

调用 MCP 协议工具，实现与外部系统的集成

3
Sub-workflow

嵌套执行其他工作流，实现复杂业务逻辑
